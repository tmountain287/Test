# 과제 이외 추가된 내용
## Localization
  - Lacalization Package(Version 1.3.2)
  - 영어, 일본어, 한국어를 구현
  - 언어 변경이 가능한 세팅 팝업 작업.
  - 선택된 언어값은 로컬에 저장.
  - IntroScene, LobbyScene에 세팅팝업 오픈 버튼 우측상단에 추가.
